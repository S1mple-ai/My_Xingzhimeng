# 智能待办事项管理系统

一个功能完整的现代化待办事项管理应用，基于 Flask + SQLite + 原生 JavaScript 构建，具有美观的UI设计和丰富的功能。

## ✨ 功能特性

### 核心功能
- ✅ 任务创建、编辑、删除
- ✅ 任务完成状态切换
- ✅ 拖拽排序
- ✅ 批量操作（批量完成、批量删除）
- ✅ 实时统计面板

### 高级功能
- 🏷️ **分类管理** - 创建自定义分类，为任务分组
- 🎯 **优先级设置** - 高、中、低三级优先级
- 📅 **日期管理** - 开始日期和截止日期
- 🔍 **搜索过滤** - 按内容、分类、状态、优先级过滤
- 📱 **响应式设计** - 完美适配手机、平板、桌面

### 用户体验
- 🎨 现代化UI设计
- ⚡ 流畅的动画效果
- 🌈 直观的颜色编码
- 📊 可视化统计信息
- 🔄 实时数据同步

## 🛠️ 技术栈

- **后端**: Python Flask + SQLAlchemy
- **数据库**: SQLite (支持 PostgreSQL/MySQL)
- **前端**: HTML5 + CSS3 + 原生 JavaScript
- **API**: RESTful API 设计
- **部署**: 支持 Heroku、Vercel、VPS 等多种部署方式

## 🔌 API 接口

### 任务管理
- `GET /api/tasks` - 获取任务列表（支持搜索和过滤）
- `POST /api/tasks` - 创建新任务
- `PUT /api/tasks/<id>` - 更新任务
- `DELETE /api/tasks/<id>` - 删除任务

### 分类管理
- `GET /api/categories` - 获取分类列表
- `POST /api/categories` - 创建新分类
- `DELETE /api/categories/<id>` - 删除分类

### 统计信息
- `GET /api/stats` - 获取任务统计数据

## 🎯 使用指南

### 创建任务
1. 在"添加新任务"区域输入任务内容
2. 选择优先级（高/中/低）
3. 选择分类（可选）
4. 设置开始日期和截止日期（可选）
5. 点击"添加任务"按钮

### 管理分类
1. 在"分类管理"区域输入分类名称
2. 选择分类颜色
3. 点击"+"按钮创建分类

### 搜索和过滤
- 使用搜索框按关键词搜索任务
- 使用下拉菜单按分类、状态、优先级过滤
- 多个过滤条件可以组合使用

### 批量操作
1. 勾选要操作的任务
2. 使用批量操作按钮进行批量完成或删除
